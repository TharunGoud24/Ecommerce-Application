Ecommerce Application
Description

This is Ecommerce web application built to provide a seamless shopping experience.
It allows users to browse products, search for items, manage a shopping cart, and perform CRUD operations on products. The project consists of:

Frontend: A responsive web interface built with modern JavaScript frameworks (e.g., React/Vite).

Backend: A RESTful API server handling product management, cart operations, and search functionality.

This project demonstrates core e-commerce features, including product management, shopping cart functionality, and search capabilities.

Features
User Features

Browse products with descriptions

Search for products by name or category

Add products to the shopping cart

Update product quantities in the cart

Remove items from the cart

Admin/Product Management Features

Add new products with details (name, description, price, stock, image)

Update existing product details

Delete products from the catalog
